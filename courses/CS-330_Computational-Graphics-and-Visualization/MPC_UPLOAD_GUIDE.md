# NEKUSAR OVERWATCH PROXY - MPC UPLOAD GUIDE

## COMPLETE STEP-BY-STEP INSTRUCTIONS

---

## PART 1: PREPARE YOUR IMAGES (Do this first!)

### Option A: Quick Method (Recommended - 30 mins)
1. Go to https://alphacoders.com/overwatch-wallpapers
2. Search for each character from the DOWNLOAD_INSTRUCTIONS.txt
3. Download 4K images (click image → Download button)
4. Rename files to match the format: `001_Nekusar the Mindrazer.jpg`

### Option B: Batch Download (Advanced - 15 mins)
1. Use a browser extension like "Download All Images" or "Image Downloader"
2. Visit character pages and batch download
3. Rename using bulk rename tool (like "Bulk Rename Utility" on Windows)

### Image Quality Checklist:
- ✓ Resolution: Minimum 1920x1080 (4K preferred)
- ✓ Format: JPG or PNG
- ✓ File size: 500KB - 5MB per image
- ✓ Character centered and visible
- ✓ Correct filename format

---

## PART 2: UPLOAD TO MAKEPLAYINGCARDS.COM

### Step 1: Go to MPC Website
**URL:** https://www.makeplayingcards.com

### Step 2: Start Your Project
1. Click "Start Your Design"
2. Select "Playing Cards" category
3. Choose "Poker Size" (2.5" × 3.5")
4. Select quantity: **100 cards**
5. Click "Start Design"

### Step 3: Upload Card Fronts
1. Click "Upload Images" button
2. Navigate to your `nekusar_overwatch_images/` folder
3. Select ALL 100 images (Ctrl+A or Cmd+A)
4. Click "Open" to upload
5. Wait for all images to upload (may take 5-10 minutes)

### Step 4: Assign Images to Cards
**METHOD 1: Drag and Drop**
- Drag each image from the left panel to a card slot
- Match numbered filenames to card positions

**METHOD 2: Auto-Fill (Faster)**
- If images are numbered 001-100, click "Auto-Fill"
- MPC will automatically assign images in order

### Step 5: Design Card Backs
**Option A: Same Back for All Cards**
1. Click "Same design for all backs"
2. Upload an Overwatch logo or Nekusar/Reaper themed image
3. Click "Apply to all backs"

**Option B: Different Backs**
- Upload 100 different back images
- Assign individually (not recommended - takes forever)

**Recommended Back Image Sources:**
- Overwatch logo: Search "Overwatch logo 4K" on Google Images
- Reaper themed: Use dark Reaper artwork
- Simple: Solid color with small Overwatch logo

### Step 6: Review and Adjust
1. Click through each card preview
2. Check that images are centered
3. Adjust cropping if needed (use "Edit" button)
4. Ensure no important parts are cut off

### Step 7: Add to Cart
1. Click "Next" or "Add to Cart"
2. Choose:
   - Card Stock: **Premium (310gsm)**
   - Finish: **Matte** (less glare) or **Glossy** (more vibrant)
   - Packaging: Standard box
3. Review total cost (should be ~$28-35)

### Step 8: Shipping
1. Enter shipping address
2. Select shipping method:
   - **Standard:** 10-15 business days (~$8-10)
   - **Express:** 5-7 business days (~$15-20)
3. Total should be: **$36-55 depending on options**

### Step 9: Checkout
1. Review order summary
2. Enter payment information
3. **IMPORTANT:** Double-check quantity (100 cards)
4. Place order!

## TROUBLESHOOTING

### "Upload failed"
- **Solution:** Reduce image file size using online compressor
- Try: https://tinypng.com or https://compressor.io

### "Image too small"
- **Solution:** Find higher resolution version
- Minimum: 1500x2100 pixels for best quality

### "Can't find certain character"
- **Solution:** Use similar character or generic Overwatch art
- Example: Multiple Reaper cards can use different Reaper poses

### "Too many cards to upload manually"
- **Solution:** MPC has bulk upload tool
- Contact their support for CSV import option
- Or use "Auto-Fill" feature if images are numbered
