# MAT-243: Applied Statistics for STEM

## Course Description

This course provides a comprehensive introduction to statistical analysis and its applications in science, technology, engineering, and mathematics (STEM) fields. Using Python and Jupyter notebooks, students learn to perform statistical analyses, interpret data, create visualizations, and communicate findings through professional summary reports. The course emphasizes practical application of statistical methods to real-world datasets and scenarios.

## Key Topics

- **Descriptive Statistics**: Measures of central tendency, variability, and distribution
- **Probability Theory**: Probability distributions, expected values, and statistical inference
- **Hypothesis Testing**: T-tests, ANOVA, chi-square tests, and p-values
- **Regression Analysis**: Linear regression, correlation, and predictive modeling
- **Data Visualization**: Creating professional charts, graphs, and statistical plots
- **Confidence Intervals**: Computing and interpreting confidence intervals
- **Statistical Inference**: Drawing conclusions from sample data
- **Data Analysis Tools**: Python libraries for statistical computing

## Tools & Technologies

- **Programming Language**: Python 3.x
- **Development Environment**: Jupyter Notebook for interactive analysis
- **Statistical Libraries**:
  - NumPy for numerical computing
  - Pandas for data manipulation
  - Matplotlib for data visualization
  - SciPy for statistical functions
  - Seaborn for advanced visualizations
- **Documentation**: Microsoft Word for summary reports
